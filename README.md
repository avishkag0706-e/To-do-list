# 📝 To-Do List Web App

A responsive to-do list web application built using HTML, CSS, and JavaScript.  
It helps users manage daily tasks efficiently with a clean and user-friendly interface.

---

## 🚀 Features
- ➕ Add new tasks
- ❌ Delete tasks
- ✅ Mark tasks as completed
- 🧹 Clear all tasks
- 📱 Responsive design (mobile & desktop friendly)

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

## 🎯 Purpose
This project was created to practice JavaScript functionality, DOM manipulation, and building interactive user interfaces.

---

## 📂 How to Use
1. Clone the repository
2. Open `index.html` in your browser
3. Add and manage your tasks
